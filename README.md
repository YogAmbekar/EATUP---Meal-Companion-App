# EATUP: Corporate Meal Companionship Platform

## Overview

**EATUP** is a mobile application designed to bridge the social gap in corporate environments. The project addresses the common issue of workplace loneliness by connecting office employees and interns for shared meals. By matching users based on location, schedule, and dietary preferences, EATUP transforms a solitary lunch break into a networking and social opportunity.

The project includes:

* **UI/UX Design & Prototyping**
* **User Flow & Journey Mapping**
* **Feature Prioritization & Requirement Analysis**
* **Market Problem Identification**
* **Success Metric Definition**

---

## Dataset & Research

The project is built upon user persona research and behavioral data within IT parks and corporate hubs, focusing on:

* **Employee Demographics:** Age, job roles, and work locations.
* **Meal Habits:** Preferred lunch timings, cuisine types, and dietary restrictions (Veg/Non-Veg).
* **Pain Points:** Frequency of eating alone and hesitation levels in social interaction.

### Project Highlights

* **Target Audience:** Corporate Professionals & IT Interns
* **Core Focus:** Reducing workplace loneliness and enhancing social bonding.
* **Key Constraint:** Professional and safe environment matching.

## Project Workflow

### 1. Problem Identification

* Conducted research on the "loneliness epidemic" in high-pressure corporate environments.
* Identified that existing apps focus on dating, which is inappropriate for professional meal breaks.

### 2. Solution Architecting

* Defined a matching algorithm based on three pillars: **Real-time Location**, **Synchronized Timing**, and **Dietary Compatibility**.

### 3. User Flow Design

* Mapped the seamless journey from app installation to post-meal feedback.
* Developed flows for both 1-to-1 matching and "Group Joins" (e.g., specific food interest groups).

### 4. UI/UX Prototyping

* Created high-fidelity screens for:
* **Smart Matching Interface:** Real-time discovery of nearby colleagues.
* **Preference Filters:** Easy selection of meal types and timings.
* **Secure Chat:** In-app communication to finalize meet-up spots.
* **Notifications:** Invitations for 1-to-1 or group meals.



### 5. Feature Prioritization

* Integrated safety features such as professional profile verification and a feedback/rating system to maintain community quality.

---

## Key Features & Results

* **Location-Based Discovery:** Uses real-time data to find companions within the same IT park or office building.
* **Schedule Sync:** Eliminates the "mismatched lunch break" problem by filtering users available at the same time.
* **Interest-Based Groups:** Created "Foodie Circles" (e.g., "Pizza Lovers") to encourage team-like bonding.
* **Increased Social Connectivity:** Designed to improve employee mental well-being and networking frequency.
